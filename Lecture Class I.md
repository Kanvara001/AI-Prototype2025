## ЁЯФН 2. Listing & Permissions (р╕Бр╕▓р╕гр╣Ар╕гр╕╡р╕вр╕Бр╕Фр╕╣р╣Др╕Яр╕ер╣М)
р╕Др╕│р╕кр╕▒р╣Ир╕З `ls` (List) р╣Гр╕Кр╣Йр╕кр╕│р╕лр╕гр╕▒р╕Ър╕Фр╕╣р╕гр╕▓р╕вр╕Бр╕▓р╕гр╣Др╕Яр╕ер╣Мр╣Бр╕ер╕░р╕гр╕▓р╕вр╕ер╕░р╣Ар╕нр╕╡р╕вр╕Фр╕кр╕┤р╕Чр╕Шр╕┤р╣Мр╕Бр╕▓р╕гр╣Ар╕Вр╣Йр╕▓р╕Цр╕╢р╕З

### ЁЯЫа р╕гр╕╣р╕Ыр╣Бр╕Ър╕Ър╕Др╕│р╕кр╕▒р╣Ир╕Зр╕Чр╕╡р╣Ир╕Щр╕┤р╕вр╕бр╣Гр╕Кр╣Й
* **`ls`** : р╣Бр╕кр╕Фр╕Зр╕гр╕▓р╕вр╕Кр╕╖р╣Ир╕нр╣Др╕Яр╕ер╣М/р╣Вр╕Яр╕ер╣Ар╕Фр╕нр╕гр╣Мр╕Чр╕▒р╣Ир╕зр╣Др╕Ы
* **`ls -l`** : р╣Бр╕кр╕Фр╕Зр╣Бр╕Ър╕Ъ **Long Format** (р╕гр╕▓р╕вр╕ер╕░р╣Ар╕нр╕╡р╕вр╕Фр╕Др╕гр╕Ър╕Цр╣Йр╕зр╕Щ)
    > `ЁЯФР Permission | ЁЯСд Users | ЁЯУж Size | ЁЯХТ Date | ЁЯУД File name`
* **`ls -lh`** : р╣Бр╕кр╕Фр╕Зр╕Вр╕Щр╕▓р╕Фр╣Др╕Яр╕ер╣Мр╣Бр╕Ър╕Ъ **Human-readable** (р╣Ар╕Кр╣Ир╕Щ 2K, 15M, 1G)
* **`ls -ltr`** : р╣Ар╕гр╕╡р╕вр╕Зр╣Др╕Яр╕ер╣Мр╕Хр╕▓р╕бр╣Ар╕зр╕ер╕▓р╕Чр╕╡р╣Ир╣Бр╕Бр╣Йр╣Др╕Вр╕ер╣Ир╕▓р╕кр╕╕р╕Ф р╣Вр╕Фр╕вр╣Др╕Яр╕ер╣М **р╣Гр╕лр╕бр╣Ир╕Чр╕╡р╣Ир╕кр╕╕р╕Фр╕Ир╕░р╕нр╕вр╕╣р╣Ир╕ер╣Ир╕▓р╕Зр╕кр╕╕р╕Ф**
    * `-t` : Sort by time
    * `-r` : Reverse order

### ЁЯФР р╕Др╕зр╕▓р╕бр╕лр╕бр╕▓р╕вр╕Вр╕нр╕З Permissions
р╕Хр╕▒р╕зр╕нр╕вр╣Ир╕▓р╕З: `drwxr-xr-x`
* `d` : **Directory** (р╕Цр╣Йр╕▓р╣Ар╕Ыр╣Зр╕Щ `-` р╕Др╕╖р╕нр╣Др╕Яр╕ер╣Мр╕Шр╕гр╕гр╕бр╕Фр╕▓)
* `r` : **Read** (р╕нр╣Ир╕▓р╕Щ)
* `w` : **Write** (р╣Ар╕Вр╕╡р╕вр╕Щ/р╣Бр╕Бр╣Йр╣Др╕В)
* `x` : **Execute** (р╕кр╕▒р╣Ир╕Зр╕гр╕▒р╕Щр╣Вр╕Ыр╕гр╣Бр╕Бр╕гр╕б)

## ЁЯЧВя╕П 3. File & Folder Management (р╕Бр╕▓р╕гр╕Ир╕▒р╕Фр╕Бр╕▓р╕гр╣Др╕Яр╕ер╣М)
